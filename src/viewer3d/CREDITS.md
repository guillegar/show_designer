# Créditos del Viewer 3D

## Dependencias

### Three.js
- **Repositorio**: https://github.com/mrdoob/three.js
- **Licencia**: MIT
- **Uso**: rendering 3D core (geometrías, materiales, OrbitControls, postproceso)

## Código del viewer

Todo el código del viewer3d (`main.js`, `moving_head.js`, shaders del beam,
escena, etc.) es **original, escrito desde cero** para el modelo de fixtures
de este proyecto. No deriva de ningún otro software.
