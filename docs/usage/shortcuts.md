# Keyboard Shortcuts ⌨️

Essential keyboard shortcuts for fast workflow.

## Playback

| Key | Action |
|-----|--------|
| **Space** | Play / Pause |
| **S** | Stop (reset to start) |
| **B** | Blackout (all lights off) |

## Editing

| Key | Action |
|-----|--------|
| **Ctrl+S** | Save show |
| **Ctrl+O** | Open show |
| **Ctrl+Z** | Undo |
| **Ctrl+Shift+Z** | Redo |
| **Ctrl+C** | Copy clip(s) |
| **Ctrl+V** | Paste |
| **Ctrl+X** | Cut |
| **Delete** | Delete selected clip |

## Tools

| Key | Action |
|-----|--------|
| **D** | Draw mode (create clips) |
| **C** | Cut/Slice mode |
| **Escape** | Select mode |
| **Q** | Toggle snap to beats/bars |

## View & Navigation

| Key | Action |
|-----|--------|
| **+** | Zoom in |
| **-** | Zoom out |
| **Ctrl+0** | Zoom to fit |
| **Home** | Go to start |
| **End** | Go to end |
| **Space** *(while paused)* | Scrub (hold to drag time) |

## Cues (Hotkeys)

| Key | Action |
|-----|--------|
| **Ctrl+0...9** | Jump to cue 0-9 |
| **Shift+0...9** | Set cue at current time |

## Tabs

| Key | Action |
|-----|--------|
| **Ctrl+1** | Timeline tab |
| **Ctrl+2** | Feedback tab |
| **Ctrl+3** | Patch tab |
| **Ctrl+4** | Analyzer tab |
| **Ctrl+M** | Analyzer (mnemonic: Music) |

## Clip Selection

| Key | Action |
|-----|--------|
| **Click** | Select single clip |
| **Shift+Click** | Add to selection |
| **Ctrl+Click** | Toggle selection |
| **Ctrl+A** | Select all clips |

## Lock / Unlock

| Key | Action |
|-----|--------|
| **Ctrl+L** | Lock selected clips (protect from editing) |
| **Ctrl+U** | Unlock selected clips |

---

**Pro tip**: Combine shortcuts for speed:
- `Ctrl+C` → `Ctrl+V` → `Ctrl+Z` to quickly duplicate and undo
- `D` → Click effect → Drag to create clips fast
- Space to play, `Ctrl+Z` to undo if it looks wrong
